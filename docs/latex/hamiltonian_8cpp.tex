\doxylink{class_hamiltonian}{Hamiltonian} class implementation

This file implements the eigensystem method for the \doxylink{class_hamiltonian}{Hamiltonian} class. The eigensystem method computes the eigenvalues and eigenvectors of the \doxylink{class_hamiltonian}{Hamiltonian} matrix H(k) at a given wavevector k using Eigen\textquotesingle{}s Self\+Adjoint\+Eigen\+Solver. 

The \doxylink{class_hamiltonian}{Hamiltonian} matrix is expected to be Hermitian, and the method will throw an exception if the eigensystem computation fails.
